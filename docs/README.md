# Documentation Index

Welcome to the Discrete Event Simulation documentation.

## Application Modes

The application supports two simulation modes:
- **Solo Run** (`pages/solo_run.py`): Single simulation with full visualization
- **Scenarios** (`pages/scenarios.py`): Multi-scenario simulations (separate documentation)
